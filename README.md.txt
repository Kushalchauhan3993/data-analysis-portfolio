# Data Analysis Portfolio

## Overview

This repository contains a collection of **five end-to-end data analysis projects** across multiple domains.
Each project demonstrates skills in **data cleaning, exploratory analysis, statistical methods, data visualization, and business insight generation** using Python.

The portfolio showcases the ability to analyze real-world datasets and extract meaningful insights using tools such as **Pandas, NumPy, Matplotlib, Seaborn, and Jupyter Notebook**.

# Portfolio Projects

## 1. Supermarket Sales Analysis (Retail Domain)

**Objective:**
Analyze supermarket transaction data to identify sales patterns, top-performing products, and customer behavior.

**Key Tasks**

* Data cleaning and preprocessing
* Revenue analysis by product category
* Customer purchasing pattern analysis
* Daily sales trend analysis

**Visualizations**

* Sales by product line
* Branch performance comparison
* Customer spending distribution
* Correlation heatmap

**Insights**

* Identified top-performing product categories
* Detected peak sales periods
* Provided recommendations for inventory optimization

---

## 2. Student Performance Analysis (Education Domain)

**Objective:**
Evaluate academic performance trends and determine the impact of study time and other factors on student outcomes.

**Key Tasks**

* Pass/Fail rate analysis
* Subject performance comparison
* Study time vs academic score correlation

**Visualizations**

* Pass vs Fail distribution
* Average subject scores
* Study time vs performance scatter plot
* Score distribution analysis

**Insights**

* Higher study time correlates with improved academic performance
* Identified subjects where students perform better or struggle
